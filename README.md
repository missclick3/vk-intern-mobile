## Описание
Реализует некоторые ручки из https://pokeapi.co/
### Реализованный функционал
- Отображение списка покемонов с подгрузкой картинок
- Отображение информации о покемоне по клику со следующими полями:
   - Изображение
   - Параметры
   - Статы
   - Способности
